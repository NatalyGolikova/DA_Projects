АНАЛИЗ ДАННЫХ О ФИЛЬМАХ 

Датасет с данными о 5000 фильмах, имеющих наибольший рейтинг на сайте IMDb 
(датасет выложен на платформе Kaggle).


Цель исследования - проанализировать данные о фильмах:
1. исследовать продолжительность фильмов
2. оценить доходность фильмов
3. проанализировать фильмы по рейтингу BBFC 
4. проанализировать данные о режиссерах представленных фильмов


Анализ включает в себя такие разделы:
1. предобработка данных (работа с пропусками, дубликатами, изменение формата данных)
2. анализ данных (количественных, категориальных)
3. графический анализ взяимосвязей между признаками
